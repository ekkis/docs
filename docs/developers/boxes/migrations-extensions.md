
migrations-extensions 
====================




## Dependencies
### Boxes
* [`localenv-extensions`](localenv-extensions.md)
* [`seed-eos`](seed-eos.md)
* [`zeus-eos`](zeus-eos.md)




## Install
```bash
zeus unbox migrations-extensions
```
## Examples
### Migrate 
```bash
zeus migrate
```
### No reset 
```bash
zeus migrate --no-reset
```
